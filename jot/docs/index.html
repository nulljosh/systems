jot

A small scripting language with a Python reference interpreter and a C99 production interpreter.

GitHub README
Syntax

Brace-delimited blocks, optional semicolons, `//` and `#` comments.

Implementations

`python/` for reference semantics, `src/` for the native interpreter.

Runtime

Functions, classes, arrays, objects, imports, file I/O, JSON, and exceptions.

Quick Start

PYTHONPATH=. python3 python/main.py examples/hello.jot
make -C src/
src/jot examples/hello.jot
make -C src/ test

Example

let name = "world"
print "hello ${name}"

fn greet(who, greeting = "hi") {
    return "${greeting}, ${who}!"
}

print greet("jot")

Highlights

Core Language

Variables, functions, classes, loops, conditionals, and default parameters.

Data Model

Numbers, strings, bools, null, arrays, objects, and callable values.

Standard Library

String, array, math, object, JSON, and file utilities built into the runtime.

Tooling

Python tests in `python/tests/` and native interpreter tests in `src/tests/`.
