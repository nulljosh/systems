# physics-engine

![version](https://img.shields.io/badge/version-v0.1.0-blue)

2D rigid body physics engine built from scratch in C. Broad/narrow phase collision detection, constraint solver, Verlet integration.

## Build

```bash
make
./physics_demo
```

## Test

```bash
make test
```
