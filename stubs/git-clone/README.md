# git-clone

![version](https://img.shields.io/badge/version-v0.1.0-blue)

Git version control system internals implemented from scratch in C. Object store, index, refs, packfiles, diff, and merge.

## Build

```bash
make
./mygit init
./mygit add file.txt
./mygit commit -m "initial"
```

## Test

```bash
make test
```
