# Text Editor - Claude Notes

## Overview
Terminal text editor built with Python + curses. nano/vim inspired. Done/stable.

## Run
```bash
cd ~/Documents/Code/text-editor
python editor.py [filename]
```

## Features
- curses-based terminal UI
- File open, edit, save
- Line numbers, status bar, command bar
- Search and replace
- Undo/redo stack

## Status
Done/stable. Systems/learning project.
