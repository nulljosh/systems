# Shell - Claude Notes

## Overview
Unix shell built from scratch. fork/exec/wait, pipes, redirects, env vars, builtins. Done/stable.

## Build & Run
```bash
cd ~/Documents/Code/shell
make
./shell
```

## Features
- REPL with prompt and line editing
- Pipes (`|`), redirects (`>`, `<`, `>>`)
- Builtins: `cd`, `exit`, `export`, `history`
- Job control (`&`, `fg`, `bg`)
- Environment variable expansion

## Status
Done/stable. Systems/learning project.
